# Contribuire a DomeOS

Grazie per il tuo interesse a contribuire a **DomeOS**! In questa cartella troverai tutta la documentazione necessaria per iniziare.

## Guide disponibili

- `howto/pull.md` — Come creare una Pull Request.
- `howto/fork.md` — Come effettuare un fork del repository.
- `build.md` — Come compilare (buildare) DomeOS.

Se trovi un bug o hai un'idea per migliorare il progetto, sentiti libero di aprire una Issue o una Pull Request.
