## Cos'è DomeOS

**DomeOS** è un progetto di sistema operativo sperimentale realizzato completamente in **Scratch**.

L'obiettivo del progetto è creare un'esperienza simile a un vero sistema operativo, permettendo di esplorare concetti come interfacce grafiche, applicazioni, gestione dei file e sviluppo software.

## Requisiti

Per utilizzare e modificare DomeOS sono necessari:

- ✅ **Scratch 3.20 o superiore**
- ✅ Un sistema operativo moderno (Windows, macOS o Linux)
- ✅ Git (consigliato per contribuire)

Se non hai Scratch installato, puoi scaricarlo dal [sito ufficiale di Scratch](https://scratch.mit.edu/download).

DomeOS utilizza un sistema di compilazione per preparare il progetto all'utilizzo e alla modifica.

Dopo aver clonato la repository, è necessario compilare DomeOS prima di poterlo utilizzare.

Per la guida completa visita:

